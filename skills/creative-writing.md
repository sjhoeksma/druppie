## Description
This is a core LLM capability used to synthesize creative text based on requirements.

## Usage
- Generating Video Scripts (`av_script`).
- Writing Voiceover Dialogue (`audio_text`).
- Describing Scenes (`visual_prompt`).

## Best Practices
- Use a specific Tone (e.g. "Educational", "Humorous").
- adhere to specified lengths/durations.
- Ensure outputs are structured (JSON) when required for downstream processing.
