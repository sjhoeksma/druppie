# Skills Overview

This directory contains the **Skill Definitions** for the Druppie platform.
Skills are specialized capabilities that Agents possess. Each skill file defines the **operational contract**, **standards**, and **processing logic** that an Agent must follow when applying that skill.

## Available Skills

| Skill | Description | Version |
| :--- | :--- | :--- |
| **[Golang](./golang.md)** | Expert in Golang programming and microservices. | 1.0.0 |
| **[Kubernetes](./kubernetes.md)** | Expert in Kubernetes operations and manifests. | 1.0.0 |
| **[Mermaid](./mermaid.md)** | Expert in generating Mermaid diagrams. | 1.0.0 |
| **[Node.js](./nodejs.md)** | Expert in Node.js backend development. | 1.0.0 |
| **[Python](./python.md)** | Expert in Python data science and scripting. | 1.0.0 |

## Usage
Skills are referenced by Agents in their configuration or adopted dynamically.
